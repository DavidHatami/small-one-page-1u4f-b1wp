# small-one-page-1u4f

A small one-page marketing site for Devin Larkin, a freelance copywriter. The site features a hero section, value propositions highlighting Devin's approach to copywriting, and a call-to-action for potential clients to get in touch.

## Local Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

Requires Node.js >= 20.

## Deploy to Netlify

1. Push your repository to GitHub, GitLab, or Bitbucket
2. Log in to [Netlify](https://netlify.com) and click "Add new site" > "Import an existing project"
3. Connect your repository
4. Configure build settings:
   - **Build command:** `npm run build`
   - **Publish directory:** `dist`
5. Click "Deploy site"

Netlify will automatically rebuild and deploy on every push to your main branch.
